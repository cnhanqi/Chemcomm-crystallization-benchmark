# Notice

This repository is a scholarly data and code release supporting a ChemComm manuscript. If you use, modify, redistribute or build upon the processed data, source tables, scripts or figures, please cite the archived release DOI and the associated article.

The processed data and figure source files are shared to support transparency, reproducibility and reuse. The included `CITATION.cff` file provides citation metadata, and the Zenodo archive DOI should be used as the version-of-record citation after release.

The underlying crystallographic records are public Protein Data Bank records. This repository does not claim ownership over the original PDB records; it provides processed benchmark tables, analysis code and derived outputs generated for this study.
